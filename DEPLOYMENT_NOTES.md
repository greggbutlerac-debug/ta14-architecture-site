# Deployment Notes

1. Back up the current repository.
2. Upload/copy all files from `public/` into the repo public directory.
3. Commit and deploy on Netlify.
4. Test these routes:
   - `/`
   - `/architecture.html`
   - `/services.html`
   - `/resources.html`
   - `/request-evaluation.html`
   - `/request`
5. Test the email CTA and Netlify form.

The `_redirects` file is included to prevent the Request Review button from producing a 404.
